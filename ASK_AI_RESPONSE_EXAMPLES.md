## üéì KEY TAKEAWAYS

### What Makes Good Responses:

1. **Answer what was asked** - Don't deflect to "here's what to do"
2. **Use specific numbers** - "47 viewers" not "many people"
3. **Quote actual comments** - Show real viewer language
4. **Add context naturally** - Not forced "action items"
5. **Adapt to question type** - Info for info, strategy for strategy

### Response Styles by Question:

| Question Type | Response Style | Include Actions? |
|--------------|----------------|------------------|
| "What did people say about X?" | Direct, specific, with examples | Optional (if natural) |
| "Is this a hit?" | Evidence-based analysis | Optional (if helpful) |
| "What should I do?" | Action-focused, prioritized | Yes (that's the point) |
| "What's interesting?" | Pattern surfacing | Optional (implications) |
| "Who should I reply to?" | Prioritized list | Yes (engagement plan) |
